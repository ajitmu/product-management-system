# Product Management System

A full-stack web application for managing products, categories, suppliers, and orders.

---

## Tech Stack

* **Frontend:** React (Vite)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Deployment:** Render

---

## Features

* Authentication (Login/Register)
* Product Management (CRUD)
* Category & Supplier Management
* Order Management
* Dashboard

---

## Project Structure

```
frontend/    # React application  
server/      # Backend API  
```

---

## Getting Started

### Backend

```
cd server
npm install
npm start
```

### Frontend

```
cd frontend
npm install
npm run dev
```
